# ToponSdkUPload

AnyThink Core SDK for Android

## Installation

Add the JitPack repository to your `build.gradle`:

```gradle
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```

Add the dependency:

```gradle
dependencies {
    implementation 'com.github.hkhatuwal:ToponSdkUPload:6.4.30'
}
```
